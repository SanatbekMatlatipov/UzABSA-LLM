# =============================================================================
# Troubleshooting: PyTorch Installation
# =============================================================================
#
# Problem: CUDA not found (torch.cuda.is_available() returns False)
# ----------------------------------------------------------------
# This usually means you installed CPU-only PyTorch instead of CUDA-enabled.
#
# Quick fix:
#   1. Uninstall existing PyTorch:
#      pip uninstall torch torchvision torchaudio -y
#
#   2. Reinstall with CUDA 12.4 (use the --index-url flag):
#      pip install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cu124
#
#   3. Verify:
#      python -c "import torch; print(f'CUDA: {torch.cuda.is_available()}, GPUs: {torch.cuda.device_count()}')"
#      Should print: CUDA: True, GPUs: {your_gpu_count}
#
#
# Problem: Torch version mismatch (e.g., torch 2.10.0 but torchaudio 2.6.0)
# -------------------------------------------------------------------------
# Happens when you partially installed CUDA packages without using --index-url.
#
# Quick fix:
#   1. Uninstall all PyTorch packages:
#      pip uninstall torch torchvision torchaudio -y
#
#   2. Reinstall all at once with CUDA 12.4:
#      pip install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cu124
#
#   Note: Install all three (torch, torchvision, torchaudio) in one command
#         to ensure version compatibility.
#
#
# Problem: "ERROR: pip's dependency resolver does not currently account..."
# -------------------------------------------------------------------------
# This warning typically appears after successful installation. 
# It can usually be ignored if torch.cuda.is_available() returns True.
# However, if you see xformers or other package conflicts, uninstall them:
#
#   pip uninstall xformers -y
