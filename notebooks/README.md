# Notebooks

This directory contains Jupyter notebooks for exploration, training, and evaluation.

## Notebooks

1. **01_data_exploration.ipynb** - Explore the Uzbek ABSA dataset
2. **02_training.ipynb** - Interactive training notebook
3. **03_evaluation.ipynb** - Evaluation and analysis

## Usage

Start Jupyter:
```bash
jupyter notebook notebooks/
```

Or use in PyCharm/VS Code with Jupyter support.
